🌈 Infinite Animated Border 

Este projeto apresenta um componente de card moderno onde as bordas ganham vida com uma animação de gradiente infinito ao interagir com o mouse. É uma técnica avançada de **CSS puro** que utiliza camadas e animações de rotação para criar um efeito visual impressionante.

## 🎯 Objetivo
Explorar o uso de **pseudo-elementos (`::before` / `::after`)** combinados com `conic-gradient` e `@keyframes` para criar bordas dinâmicas que não sobrecarregam a performance da página.

## ✨ Diferenciais Técnicos
- **Animação de Gradiente:** Rotação suave de cores que cria a ilusão de movimento contínuo nas bordas.
- **Efeito de Vidro (Glassmorphism):** O centro do card mantém uma transparência elegante, destacando a animação periférica.
- **Interatividade Otimizada:** A animação é disparada pelo estado de `:hover`, economizando processamento quando o usuário não está interagindo.
- **Cores Vibrantes:** Paleta neon que se destaca em temas escuros (Dark Mode).
